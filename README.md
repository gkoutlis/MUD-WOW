# MUD-WoW: A WoW-themed Multiplayer Text Game

A learning project to master asyncio, WebSockets, concurrency, 
Docker, and CI/CD by building a multiplayer text-based MUD 
inspired by World of Warcraft.

## Learning Journey

This repo documents my step-by-step learning. Each phase has 
its own folder under `experiments/` with code, and `docs/notes/` 
contains the theory I learned along the way.
